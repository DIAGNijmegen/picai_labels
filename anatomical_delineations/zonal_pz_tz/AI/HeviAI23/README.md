# Zonal Prostate Segmentation in MRI

The zonal prostate segmentations in this directory (https://github.com/DIAGNijmegen/picai_labels/tree/main/zonal_pz_tz/AI/HeviAI23) have been generated by the algorithm from (A. Karag√∂z et al., 2022). See their [method description](https://rumc-gcorg-p-public.s3.amazonaws.com/evaluation-supplementary/642/88bd85e5-ca10-4979-875e-eba7476491d3/Hevi_picai_report.pdf) for details. The segmentations indicate the peripheral zone as label 1, and the transition zone as label 2. Softmax predictions for these cases can be found on [Zenodo](https://zenodo.org/record/7615350) (doi: 10.5281/zenodo.7615350).
